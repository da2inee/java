# 전체 자바 예시 (학습 순서별)

프로젝트에 있는 **모든 주제**를 **예시 코드**와 함께 정리한 문서예요.  
각 단계별 `.java` 파일 경로도 적어 두었으니, 직접 실행해 보면서 보면 됩니다.

---

## 0. 컴파일 과정

**흐름**: `.java` → `javac` → `.class` → JVM → 실행

```bash
javac HelloWorld.java   # HelloWorld.class 생성
java HelloWorld         # 실행
```

**예시 파일**: `00-compilation/HelloWorld.java`

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

## 2. 변수와 데이터 타입

**예시 파일**: `02-variables/Variables.java`

```java
public class Variables {
    public static void main(String[] args) {
        int age = 20;
        double height = 175.5;
        String name = "홍길동";
        boolean isStudent = true;

        System.out.println("이름: " + name);
        System.out.println("나이: " + age);
        System.out.println("키: " + height);
        System.out.println("학생? " + isStudent);
    }
}
```

| 타입     | 의미   | 예시                |
|----------|--------|---------------------|
| `int`    | 정수   | `int age = 20;`     |
| `double` | 실수   | `double h = 175.5;` |
| `String` | 문자열 | `String name = "김";` |
| `boolean`| 참/거짓| `boolean ok = true;` |

## 3. 연산자

**예시 파일**: `03-operators/Operators.java`

```java
int a = 10, b = 3;

// 산술: + - * / %
System.out.println("a + b = " + (a + b));
System.out.println("a / b = " + (a / b));
System.out.println("a % b = " + (a % b));

// 비교: == != > < >= <=
System.out.println("a > b ? " + (a > b));
System.out.println("a == 10 ? " + (a == 10));

// 논리: && || !
// (a > 0 && a < 10)  → a가 0보다 크고 10보다 작을 때 true
```

## 4. 조건문 (if, else if, else)

**예시 파일**: `04-condition/IfElse.java`

```java
int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else if (score >= 70) {
    System.out.println("C");
} else {
    System.out.println("F");
}
// 출력: B
```

**switch** 예시 (문서 참고):

```java
int day = 3;
switch (day) {
    case 1: System.out.println("월"); break;
    case 2: System.out.println("화"); break;
    default: System.out.println("그 외");
}
```

## 8. 클래스와 객체

**예시 파일**: `08-class/Person.java`

```java
public class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    void introduce() {
        System.out.println("저는 " + name + ", " + age + "살입니다.");
    }

    public static void main(String[] args) {
        Person p = new Person("김자바", 25);
        p.introduce();
    }
}
```

- **클래스** = 설계도, **객체** = `new`로 만든 것.
- **생성자**: 클래스 이름과 같고, 리턴 타입 없음. `new Person(...)` 할 때 실행.
- **this**: 이 객체 자신 (필드와 매개변수 구분할 때).

### 생성자 예시 (기본 생성자)

```java
public class Box {
    int size;
    // 생성자 없음 → 컴파일러가 Box() { } 자동 추가
}
Box b = new Box();
```

### 오버로딩 vs 오버라이딩

**오버로딩** (같은 클래스, 이름 같고 매개변수만 다름):

```java
int add(int a, int b) { return a + b; }
int add(int a, int b, int c) { return a + b + c; }
double add(double a, double b) { return a + b; }
```

**오버라이딩** (자식이 부모 메소드 덮어씀):

```java
class Animal { void speak() { System.out.println("..."); } }
class Dog extends Animal {
    @Override
    void speak() { System.out.println("멍멍"); }
}
Animal a = new Dog();
a.speak();  // "멍멍"
```

## 9. 어노테이션

**문서**: `09-annotations/어노테이션.md`

- **형태**: `@이름` 또는 `@이름(속성=값)`
- **역할**: 메타데이터. 컴파일러·프레임워크가 읽고 동작을 정함.

**자바 기본 예시**:

```java
@Override
public String toString() { ... }

@Deprecated
public void oldMethod() { }

@SuppressWarnings("unchecked")
List list = new ArrayList();
```

**Spring 예시** (요청 데이터):

```java
@RequestParam("name") String name        // ?name=값
@RequestBody UserDto user               // JSON → 객체
@PathVariable Long id                   // /users/{id}
@GetMapping("/hello")
public String hello() { return "hello"; }
```

## 10. String

**문서**: `10-string/String-면접.md`

**불변성**: 한 번 만들어진 문자열은 바꿀 수 없음. 연산하면 **새 객체** 생성.

**equals vs ==**:

```java
String a = "hello";
String b = "hello";
String c = new String("hello");

a == b;       // true  (리터럴 풀에서 같은 객체)
a.equals(b);  // true  (내용 같음)
a == c;       // false (서로 다른 객체)
a.equals(c);  // true  (내용 같음)
```

- **==**: 참조(주소) 비교.
- **equals**: 내용 비교. String은 **equals**로 비교.

**StringBuilder** (이어 붙일 때):

```java
StringBuilder sb = new StringBuilder();
sb.append("Hello").append(" ").append("Java");
String result = sb.toString();
```

## 11. 예외 처리

**예시 파일**: `11-exception/ExceptionExample.java`  
**문서**: `11-exception/예외처리.md`

### try / catch / finally

```java
try {
    int n = Integer.parseInt("abc");   // NumberFormatException
} catch (NumberFormatException e) {
    System.out.println("숫자가 아님: " + e.getMessage());
} finally {
    System.out.println("-- 끝 --");
}
```

### 예외 종류별 예시

| 예외 | 상황 | 예시 |
|------|------|------|
| NumberFormatException | 문자열→숫자 실패 | `Integer.parseInt("abc")` |
| ArrayIndexOutOfBoundsException | 잘못된 배열 인덱스 | `arr[10]` (길이 3일 때) |
| NullPointerException | null 참조 | `str.length()` (str이 null) |
| ArithmeticException | 0으로 나눔 | `10 / 0` |

### 여러 catch, throws

```java
try {
    // ...
} catch (NumberFormatException e) { ... }
  catch (Exception e) { ... }

public void readFile() throws IOException {
    Files.readAllLines(Path.of("file.txt"));
}
```

## 12. 컬렉션 (List, Set, Map)

**예시 파일**: `12-collection/CollectionExample.java`  
**문서**: `12-collection/컬렉션.md`

### List (ArrayList) – 순서 O, 중복 O

```java
List<String> names = new ArrayList<>();
names.add("김철수");
names.add("이영희");
names.get(1);      // "이영희"
names.size();      // 2
names.remove(0);
```

### Set (HashSet) – 중복 X

```java
Set<String> ids = new HashSet<>();
ids.add("a");
ids.add("b");
ids.add("a");   // 중복 무시
// [a, b], size() = 2
```

### Map (HashMap) – 키·값 쌍

```java
Map<String, Integer> scores = new HashMap<>();
scores.put("김철수", 90);
scores.put("김철수", 95);   // 덮어씀
scores.get("김철수");       // 95
scores.getOrDefault("박민수", 0);

for (String key : scores.keySet()) {
    System.out.println(key + " : " + scores.get(key));
}
for (Map.Entry<String, Integer> e : scores.entrySet()) {
    System.out.println(e.getKey() + " : " + e.getValue());
}
```
