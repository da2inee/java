# 자바 개념 정리

각 단계에서 쓰는 개념을 쉽게 정리했습니다. 코드 보기 전에 읽어보면 좋아요.

---

## 1. Hello World – 프로그램의 시작

- **클래스(class)**  
  자바는 모든 코드가 클래스 안에 있어야 합니다.  
  `public class HelloWorld` → "HelloWorld라는 클래스"를 만든다는 뜻.

- **main 메서드**  
  프로그램을 실행하면 **가장 먼저** 실행되는 곳입니다.  
  `public static void main(String[] args)` 형태로 한 번만 쓰면 됩니다.

- **System.out.println()**  
  괄호 안의 내용을 **한 줄** 출력합니다.  
  `System.out.print()`는 줄바꿈 없이 출력합니다.

---

## 2. 변수(Variable)와 데이터 타입

- **변수**  
  값을 담아두는 **이름 붙인 상자**라고 생각하면 됩니다.  
  나중에 그 이름으로 값을 바꾸거나 다시 쓸 수 있습니다.

- **자주 쓰는 타입**

| 타입     | 의미   | 예시                |
|----------|--------|---------------------|
| `int`    | 정수   | `int age = 20;`     |
| `double` | 실수   | `double h = 175.5;` |
| `String` | 문자열 | `String name = "김";` |
| `boolean`| 참/거짓| `boolean ok = true;` |

- **규칙**  
  - 변수는 **한 번 타입을 정하면** 그 타입만 저장합니다.  
  - 이름은 보통 소문자로 시작하고, 여러 단어면 두 번째 단어부터 대문자 (예: `myAge`).

---

## 3. 연산자(Operator)

- **산술 연산자**  
  `+` `-` `*` `/` `%`  
  `%`는 나머지 (예: `10 % 3` → 1).

- **비교 연산자**  
  `==` 같다, `!=` 다르다, `>` `<` `>=` `<=`  
  결과는 항상 **참(true) 또는 거짓(false)**.

- **논리 연산자**  
  `&&` 그리고, `||` 또는, `!` 부정  
  예: `(a > 0 && a < 10)` → a가 0보다 크고 10보다 작을 때만 true.

---

## 4. 조건문 (if, else if, else)

- **if**  
  조건이 **참일 때만** 안쪽 코드를 실행합니다.

- **else if**  
  위의 if가 거짓일 때, 이 조건을 한 번 더 검사합니다.

- **else**  
  위의 조건들이 **전부 거짓**일 때 실행합니다.

- **switch**  
  어떤 변수 값에 따라 여러 갈래 중 하나를 고를 때 씁니다.  
  if–else if를 길게 쓴 것과 비슷한 역할입니다.

---

## 5. 반복문 (for, while)

- **for**  
  "몇 번 반복할지"가 비교적 명확할 때 씁니다.  
  `for (시작; 조건; 한 번 끝날 때 할 일)`  
  예: `for (int i = 0; i < 5; i++)` → i가 0,1,2,3,4일 때 반복.

- **while**  
  "조건이 참인 동안" 계속 반복합니다.  
  조건이 처음부터 거짓이면 한 번도 실행되지 않습니다.

- **주의**  
  조건이 계속 true면 **무한 루프**가 됩니다.  
  반복문 안에서 조건이 바뀌도록 (예: 변수 증가) 꼭 넣어야 합니다.

---

## 6. 메서드(Method)

- **메서드**  
  **이름 붙인 코드 묶음**입니다.  
  필요할 때마다 그 이름을 불러서 재사용합니다.

- **void**  
  "돌려줄 값이 없다"는 뜻입니다.  
  `void sayHello()` → 출력만 하고 값은 반환하지 않음.

- **return**  
  "이 값을 돌려준다"는 뜻입니다.  
  `int add(int a, int b) { return a + b; }`  
  → 호출한 쪽에서 `add(3, 5)`를 쓰면 8을 받습니다.

- **파라미터(매개변수)**  
  메서드가 받는 값입니다.  
  `add(int a, int b)` → a, b 두 개의 정수를 받는다는 뜻입니다.

---

## 7. 배열(Array)

- **배열**  
  **같은 타입**의 값을 **순서대로** 여러 개 담는 통입니다.

- **선언과 초기화**  
  `int[] numbers = { 10, 20, 30 };`  
  또는 `int[] arr = new int[5];` → 크기 5, 값은 0으로 채워짐.

- **인덱스**  
  번호는 **0부터** 시작합니다.  
  `numbers[0]` → 첫 번째, `numbers[1]` → 두 번째.

- **length**  
  `배열이름.length` → 배열의 길이(개수)입니다.  
  반복문에서 `i < numbers.length`처럼 자주 씁니다.

---

## 8. 클래스(Class)와 객체(Object)

- **클래스**  
  객체를 만들기 위한 **설계도**입니다.  
  속성(변수)과 동작(메서드)을 정의합니다.

- **객체**  
  클래스 설계도로 만든 **실제 데이터** 하나하나입니다.  
  `new 클래스이름()`으로 생성합니다.

- **생성자(Constructor)**  
  객체가 **처음 만들어질 때** 한 번 실행되는 코드입니다.  
  클래스와 같은 이름의 메서드이고, 보통 여기서 변수 초기값을 넣습니다.

- **this**  
  "이 객체 자신"을 가리킵니다.  
  `this.name = name;` → 파라미터 name을 이 객체의 name 변수에 넣는다는 뜻입니다.

---

## 다음에 배우면 좋은 것

- **상속** – 클래스가 다른 클래스를 확장하는 것  
- **인터페이스** – 반드시 가져야 할 메서드 목록을 정해 두는 것  
- **예외 처리** – `try`, `catch`로 에러 상황 처리  
- **컬렉션** – `ArrayList`, `HashMap` 등 여러 개 데이터 다루기
