# 자바 실습 프로젝트

지금까지 정리한 **변수 → 연산자 → 조건문 → 반복문 → 메서드 → 배열 → 클래스 → String → 예외 → 컬렉션**을  
**예시 실행 + 직접 연습**으로 복습하는 폴더예요.

---

## 실행 방법

```bash
cd practice
javac JavaPractice.java
java JavaPractice
```

실행하면 **메뉴 번호**가 나옵니다. 1~10 중 하나 입력하면 해당 단계 **예시**가 실행돼요.

---

## 학습 순서 + 미니 과제

| 번호 | 주제 | 예시 실행 (JavaPractice) | 직접 해보기 (MyPractice) |
|------|------|---------------------------|--------------------------|
| 1 | 변수·타입 | int, double, String, boolean 출력 | 나이·이름 변수 만들고 출력 |
| 2 | 연산자 | 산술, 비교, 논리 | `a % 2 == 0` 으로 짝수/홀수 판별 |
| 3 | 조건문 | if / else if / else | 점수 구간별 등급 출력 (A/B/C/F) |
| 4 | 반복문 | for, while | 1~10 합 구하기 |
| 5 | 메서드 | void, return, 파라미터 | 두 수 곱하는 메서드 만들기 |
| 6 | 배열 | length, for 순회 | 배열에서 최댓값 찾기 |
| 7 | 클래스·객체 | 생성자, 메서드 | Person 예시 참고 |
| 8 | String | equals, == 차이 | 두 문자열 equals 로 "같음" 출력 |
| 9 | 예외 | try-catch | parseInt 예외 잡고 메시지 출력 |
| 10 | 컬렉션 | List, Set, Map | List에 이름 3개 넣고 for-each로 출력 |

---

## 파일 설명

| 파일 | 용도 |
|------|------|
| **JavaPractice.java** | 메뉴로 1~11 선택하면 해당 **예시** 실행. 개념 복습용. |
| **MyPractice.java** | 주석으로 "여기 채워 보세요" 적어 둔 **연습용** 클래스. 직접 코드 넣어 보기. |

---

## 추천 순서

1. **JavaPractice** 실행해서 1번(변수)부터 11번(컬렉션)까지 **한 번씩** 눌러 보기.
2. **전체-자바-예시.md** 나 **각 단계 .md** 와 같이 보면서 "이 코드가 이렇게 나오는구나" 확인.
3. **MyPractice.java** 열어서 주석 과제대로 **직접 메서드 채워 보기**.
4. 막히면 **JavaPractice** 해당 번호 다시 실행해서 예시 출력 참고.

이 폴더만 돌면서 예시 + 실습 같이 하면 돼요.
