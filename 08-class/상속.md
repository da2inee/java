# 상속 (Inheritance)

## 상속이란?
**상속**은 기존 클래스(부모)의 필드와 메소드를 새로운 클래스(자식)가 물려받아 재사용하는 것이다.

자바에서는 `extends` 키워드를 사용한다.

## 왜 필요한가?
- 공통 기능을 부모로 올려 중복 코드를 줄일 수 있다.
- 클래스 간 관계를 명확하게 표현할 수 있다.
- 기능 확장이 필요할 때 자식 클래스에서 유연하게 확장할 수 있다.

## 예시 코드
```java
class Animal {
    protected String name;

    public Animal(String name) {
        this.name = name;
    }

    public void eat() {
        System.out.println(name + "가 먹는다.");
    }
}

class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    public void bark() {
        System.out.println(name + "가 멍멍 짖는다.");
    }
}
```

## 실무에서 중요한 포인트
- "is-a 관계"일 때만 상속을 사용한다.
  - 예: `Dog is an Animal`
- 단순 재사용 목적이면 상속보다 합성(Composition)을 먼저 고려한다.
- 부모 클래스 변경은 자식 클래스에 영향을 줄 수 있으니, 공통 규약을 신중히 설계한다.

## 한 줄 정리
상속은 "공통 기능을 부모에 두고, 자식이 물려받아 확장하는 구조"다.
