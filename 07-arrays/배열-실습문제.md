# 배열 실습문제 10개

아래 문제는 `07-arrays/ArraysPractice.java`의 TODO와 1:1로 맞춰져 있습니다.
처음엔 힌트 안 보고 풀고, 막히면 힌트를 보세요.

---

## 문제 1. 배열 기본 출력
- `int[] nums = {3, 6, 9, 12};`
- 첫 번째 값, 마지막 값, 길이를 출력하세요.

힌트:
- 첫 번째: `nums[0]`
- 마지막: `nums[nums.length - 1]`

---

## 문제 2. 합계 구하기
- `nums` 전체 합계를 구해서 출력하세요.

힌트:
- `sum` 변수를 0으로 시작
- `for-each`로 누적

---

## 문제 3. 평균 구하기
- 문제 2의 합계를 이용해서 평균을 `double`로 출력하세요.

힌트:
- 정수 나눗셈 방지: `(double) sum / nums.length`

---

## 문제 4. 최댓값 찾기
- `int[] arr = {5, 2, 9, 1, 7};`
- 최댓값을 찾아 출력하세요.

힌트:
- `max = arr[0]`로 시작
- 더 큰 값 나오면 갱신

---

## 문제 5. 최솟값 찾기
- 문제 4와 같은 배열에서 최솟값을 찾아 출력하세요.

힌트:
- `min = arr[0]`로 시작
- 더 작은 값 나오면 갱신

---

## 문제 6. 짝수 개수 세기
- `arr`에서 짝수의 개수를 세어 출력하세요.

힌트:
- `if (n % 2 == 0)`이면 count++

---

## 문제 7. 값 찾기 (선형 탐색)
- `target = 7`이 배열에 있는지 찾아
  - 있으면 인덱스 출력
  - 없으면 -1 출력

힌트:
- 처음에 `index = -1`
- 찾으면 `index = i; break;`

---

## 문제 8. 배열 뒤집기 출력
- `arr`를 뒤에서부터 출력하세요. (원본 변경 X)

힌트:
- `for (int i = arr.length - 1; i >= 0; i--)`

---

## 문제 9. 2차원 배열 합계
- `int[][] matrix = {{1,2,3},{4,5,6}};`
- 전체 합계를 구해서 출력하세요.

힌트:
- 이중 for문 또는 이중 for-each

---

## 문제 10. 배열 복사
- `int[] src = {10, 20, 30};`
- 같은 길이의 `dest`를 만들고 값을 복사한 뒤 출력하세요.

힌트:
- `dest[i] = src[i]`
- `java.util.Arrays.toString(dest)` 사용 가능

---

## 체크 포인트
- `length`는 메서드가 아니라 필드 (`length()` 아님)
- 인덱스 범위는 `0` ~ `length - 1`
- 범위 벗어나면 `ArrayIndexOutOfBoundsException`
