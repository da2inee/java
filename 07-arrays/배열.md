# 배열 (Array)

배열은 **같은 타입의 값을 여러 개** 저장하는 자료구조입니다.
인덱스(index)는 **0부터 시작**합니다.

---

## 1. 선언과 초기화

```java
int[] a = new int[5];          // 길이 5, 기본값 0
int[] b = { 10, 20, 30 };      // 리터럴 초기화
String[] names = { "김", "이" };
```

- `new int[5]`는 칸만 만들고 값은 기본값으로 채워집니다.
- 기본값: `int=0`, `double=0.0`, `boolean=false`, 참조형=`null`

---

## 2. 인덱스와 length

```java
int[] nums = { 10, 20, 30 };
System.out.println(nums[0]);      // 10 (첫 번째)
System.out.println(nums.length);  // 3
```

- 첫 요소는 `nums[0]`
- 마지막 요소는 `nums[nums.length - 1]`
- `length`는 **필드**라서 `length()`가 아니라 `length`입니다.

---

## 3. 순회 방법

### (1) for 인덱스 순회

```java
for (int i = 0; i < nums.length; i++) {
    System.out.println("nums[" + i + "] = " + nums[i]);
}
```

### (2) for-each 순회

```java
for (int n : nums) {
    System.out.println(n);
}
```

- 인덱스가 필요하면 `for`
- 값만 보면 되면 `for-each`가 간단

---

## 4. 자주 하는 문제

### 합계

```java
int sum = 0;
for (int n : nums) {
    sum += n;
}
```

### 최댓값

```java
int max = nums[0];
for (int n : nums) {
    if (n > max) max = n;
}
```

---

## 5. 2차원 배열

```java
int[][] matrix = {
    { 1, 2, 3 },
    { 4, 5, 6 }
};

System.out.println(matrix[1][2]); // 6
```

- `matrix[행][열]`
- 바깥 배열 길이 = 행 개수, 안쪽 배열 길이 = 각 행의 열 개수

---

## 6. 주의할 점

- 범위를 벗어나면 `ArrayIndexOutOfBoundsException`
  - 예: 길이가 3인데 `arr[3]` 접근
- 배열은 길이가 고정이라 크기를 바꿀 수 없음
  - 크기 변경이 자주 필요하면 `ArrayList` 사용

---

## 7. 면접용 한 줄 정리

- 배열은 **같은 타입의 고정 길이 자료구조**, 인덱스는 **0부터**, 길이는 `length`로 조회합니다.
- 탐색은 보통 O(n), 인덱스 접근은 O(1)입니다.
